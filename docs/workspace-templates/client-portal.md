# Client Portal Template

**Category:** Agency  
**Best for:** Agencies, Consultants, Client delivery teams  
**Widget count:** 4

## What it includes

| Widget | Kind | Position | Purpose |
|--------|------|----------|---------|
| Client Summary | rich-text | Top-left (4×4) | Current priorities, owner notes, next milestone |
| Companies | view | Top-center (5×5) | Client company table with name + domain |
| Client Portal Embed | iframe | Top-right (3×5) | Embed a client-facing portal or Loom/Figma link |
| Delivery Health | chart | Left-mid (4×4) | Bar chart showing delivery metrics |

## How to apply

1. Open the workspace builder
2. Click **Templates** in the toolbar
3. Select **Client Portal**
4. Click **Apply to Current Tab** or **Clone as New Dashboard**

## Config shape

When applied, the canvas will contain four widgets with the positions above. All widgets use static config-backed bindings — no live data is fetched.

Edit widget settings to:
- Update the **Client Summary** rich-text body with real client notes
- Set the **Companies** source, columns, and rows to your actual company data
- Paste a real URL into **Client Portal Embed**
- Edit **Delivery Health** chart values to reflect your metrics

## Save

After editing, click **Save** to persist to `growthub.config.json`.
