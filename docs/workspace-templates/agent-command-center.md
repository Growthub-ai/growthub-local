# Agent Command Center Template

**Category:** Agent  
**Best for:** AI teams, Operators, Workspace admins  
**Widget count:** 2 (blank foundation)

## What it includes

This is a conceptual template — it does not appear in the built-in gallery yet. It describes a recommended layout for an agent-operated workspace command center.

| Widget | Kind | Suggested Position | Purpose |
|--------|------|--------------------|---------|
| Status Overview | rich-text | Top (8×3) | Active agent jobs, last run, next scheduled |
| Output Table | view | Bottom (8×6) | Recent outputs, artifacts, run IDs |

## How to create it

1. Start from the **Blank** template
2. Add a **Rich Text** widget for status notes
3. Add a **View** widget for output records
4. Save and use it as your agent monitoring surface

## Future

When Growthub Bridge is connected, these widgets can bind to live agent output data via bridge-backed bindings.
