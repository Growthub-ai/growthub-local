# Content Ops Template

**Category:** Content  
**Best for:** Editorial teams, Content marketers, Content reviewers  
**Widget count:** 3

## What it includes

| Widget | Kind | Position | Purpose |
|--------|------|----------|---------|
| Content Queue | view | Left (5×5) | Channel + status table for content pipeline |
| Publishing Mix | chart | Center (4×4) | Channel volume chart |
| Review Notes | rich-text | Right (3×4) | Open review notes and approval blockers |

## Use case

Track content by channel and status (Draft, Review, Scheduled, Published). Visualize volume distribution in the chart. Keep active review notes in the text panel.

## Config shape

Three widgets in a horizontal layout. The Content Queue uses a manual-rows binding with Channel and Status columns. Edit the rows to reflect your actual pipeline.
