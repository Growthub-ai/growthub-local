# Creative Review Template

**Category:** Creative  
**Best for:** Creative leads, Designers, Account managers  
**Widget count:** 3

## What it includes

| Widget | Kind | Position | Purpose |
|--------|------|----------|---------|
| Creative Preview | iframe | Left (7×6) | Embed a Figma, Loom, or staging page URL |
| Approval Notes | rich-text | Top-right (5×3) | Feedback, approvals, revision requests |
| Review Queue | view | Bottom-right (5×4) | Asset status table: Asset, Status |

## Use case

Set the **Creative Preview** URL to a Figma share link, a Loom recording, or a staging environment. Write approval decisions in the notes widget. Track asset status in the queue table.

## Config shape

Three widgets arranged as a large embed on the left with a notes + queue column on the right. Works well on wide displays.
