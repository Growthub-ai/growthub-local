# Reporting Dashboard Template

**Category:** Reporting  
**Best for:** Executives, Analytics teams, Operations leads  
**Widget count:** 4

## What it includes

| Widget | Kind | Position | Purpose |
|--------|------|----------|---------|
| Pipeline Trend | chart | Top-left (4×5) | Trend data as bar chart |
| Conversion | chart | Top-center (4×5) | Conversion rate chart |
| Performance Table | view | Top-right (4×5) | Key performance records |
| Executive Summary | rich-text | Bottom-left (6×3) | Weekly readout, risks, decisions |

## Use case

Drop this on a new dashboard for weekly reporting reviews. Fill in real values in the chart widgets and swap out the performance table source with your own data.

## Config shape

Four widgets, single-tab canvas. All bindings are static (no remote data). Bind to a live data source by connecting Growthub Bridge.
