# `@growthub/agent-sdk`

Thin SDK for the Growthub Agent Service generated by:

```bash
growthub starter init --seed-config served-agent --out ./growthub-agent-service
```

## Usage

```js
import { createGrowthubAgentClient } from "@growthub/agent-sdk";

const agent = createGrowthubAgentClient({
  baseUrl: process.env.GROWTHUB_AGENT_SERVICE_URL
});

const result = await agent.query("Build a governed onboarding dashboard", {
  workspace: { id: "client-portal" }
});

console.log(result.output);
```

Secrets stay outside the workspace config. If the service requires auth, set `GROWTHUB_AGENT_SERVICE_API_KEY` in the caller environment or pass `apiKey` to the client.
