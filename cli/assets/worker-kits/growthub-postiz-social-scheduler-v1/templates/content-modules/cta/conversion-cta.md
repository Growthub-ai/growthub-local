# Conversion CTA Module

**Use when:** Driving sign-ups, purchases, downloads, or link clicks.

---

## Patterns

| Pattern | Example | Platform fit |
|---|---|---|
| Link-in-bio | "Link in bio to get started." | Instagram |
| Direct link | "Try it free: [URL]" | LinkedIn, Twitter, Facebook |
| DM prompt | "DM me 'START' and I'll send you the guide." | Instagram |
| Comment trigger | "Comment 'GUIDE' and I'll send you the link." | Instagram, TikTok |
| Time-bound | "Offer ends Friday — link in bio." | All |
| Social proof CTA | "Join 10,000+ marketers already using [X]." | All |
| Command CTA | "npm create growthub-local@latest" | Twitter, LinkedIn |

## Rules

- Conversion CTAs belong on product and launch posts — not every post
- Max 20-25% of posts should have conversion CTAs (prevents audience fatigue)
- Always pair with value in the post body — earn the click
- Use UTM parameters on all links for tracking
- On Instagram: use link-in-bio tools (Linktree, etc.) since captions don't support links
- On LinkedIn and Twitter: direct links are clickable — use them
