# Value Body Module

**Use when:** Tips, how-tos, educational content, actionable advice.

---

## Structure

```text
[Context — why this matters (1-2 sentences)]

[Tip/Step 1 — specific, actionable]
[Tip/Step 2 — specific, actionable]
[Tip/Step 3 — specific, actionable]

[Key takeaway — the one thing to remember]
```

## Rules

- Every tip must be actionable — "do X" not "think about X"
- Use numbered lists or bullet points for scannability
- One idea per paragraph
- Keep sentences short on mobile-first platforms
- End with a single clear takeaway
- For carousels: one tip per slide
- For threads: one tip per tweet
