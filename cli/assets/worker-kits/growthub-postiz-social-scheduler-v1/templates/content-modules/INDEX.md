# Content Modules — Index

Modular building blocks for composing social media posts. Each module is a reusable component that can be mixed into any post draft.

---

## Hooks (opening lines)

| Module | Path | When to use |
|---|---|---|
| Curiosity hook | `hooks/curiosity-hook.md` | Educational posts, product reveals, surprising facts |
| Story hook | `hooks/story-hook.md` | Behind-the-scenes, case studies, personal lessons |
| Stat hook | `hooks/stat-hook.md` | Data-driven content, social proof, industry insights |

## Body Copy

| Module | Path | When to use |
|---|---|---|
| Value body | `body-copy/value-body.md` | Tips, how-tos, educational content |
| Story body | `body-copy/story-body.md` | Case studies, testimonials, behind-the-scenes |

## CTAs (calls to action)

| Module | Path | When to use |
|---|---|---|
| Engagement CTA | `cta/engagement-cta.md` | Polls, questions, save/share prompts |
| Conversion CTA | `cta/conversion-cta.md` | Sign-up, purchase, link-in-bio prompts |

## Composition Rules

1. Every post must have exactly one hook, one body, and one CTA.
2. Mix module types to avoid pattern fatigue (don't use the same hook type 3 posts in a row).
3. The hook type should match the content pillar (see skills.md step 5).
4. CTAs must align with the post objective from the campaign brief.
