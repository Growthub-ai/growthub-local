# QA Checklist — [CAMPAIGN NAME]

**Client:** [client-slug]
**Reviewer:** [name]
**Date:** YYYY-MM-DD

---

## Brand Voice Review

- [ ] All posts match the brand voice adjectives from the brand kit
- [ ] Platform-specific personas are respected (e.g. professional on LinkedIn, casual on TikTok)
- [ ] No prohibited phrases or claims appear in any post
- [ ] Approved CTAs are used consistently
- [ ] Tone is consistent across the campaign

## Platform Compliance Review

- [ ] Instagram captions under 2,200 characters
- [ ] LinkedIn posts under 3,000 characters
- [ ] Twitter posts under 280 characters (or marked as long-form)
- [ ] TikTok captions under 2,200 characters
- [ ] All media meets platform aspect ratio requirements
- [ ] No broken links or missing UTM parameters

## Content Pillar Audit

- [ ] Every post maps to a defined content pillar
- [ ] Pillar distribution matches the campaign brief allocation
- [ ] No orphan posts without pillar assignment
- [ ] Weekly theme arcs are coherent

## Hashtag Audit

- [ ] Hashtag count is within platform guidelines
- [ ] No prohibited or shadowbanned hashtags
- [ ] Branded hashtags are consistently included
- [ ] Niche/medium/broad tier mix is balanced
- [ ] Campaign hashtags are active for the posting period

## Scheduling Conflict Check

- [ ] No duplicate posts on the same platform at the same time
- [ ] Posting times align with optimal windows from the calendar
- [ ] Timezone is correct for the target audience
- [ ] No scheduling conflicts with holidays or sensitive dates
- [ ] BullMQ queue priority is set correctly for time-sensitive posts

## Media Asset Verification

- [ ] All referenced media assets exist and are accessible
- [ ] Image dimensions match platform requirements
- [ ] Video duration is within platform limits
- [ ] Alt text is provided for all images
- [ ] No watermarks, low-res images, or placeholder graphics

## Link and CTA Validation

- [ ] All links resolve correctly
- [ ] UTM parameters are properly formatted
- [ ] Link-in-bio is updated for traffic-driving posts
- [ ] CTAs match the intended post objective

## Handoff Sign-Off

- [ ] Execution handoff document is complete
- [ ] Postiz workspace and scheduling instructions are clear
- [ ] API payload examples are correct (if api-direct mode)
- [ ] Retry and error handling notes are included
- [ ] Analytics tracking plan is documented
