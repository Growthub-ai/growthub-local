# Social Format Library — Index

Reference this index when selecting the right format for a post. Each format template defines platform constraints, structure, and best practices.

---

## Available Formats

| Platform | Format | Template | Best for |
|---|---|---|---|
| Instagram | Feed post | `instagram-feed.md` | Product showcase, quotes, single images |
| Instagram | Carousel | `instagram-carousel.md` | Educational content, step-by-step, storytelling |
| Instagram | Reels | `instagram-reels.md` | Short video, trends, behind-the-scenes |
| TikTok | Video | `tiktok-video.md` | Short-form video, trends, tutorials |
| LinkedIn | Post | `linkedin-post.md` | Thought leadership, case studies, updates |
| Twitter/X | Post | `twitter-post.md` | Quick takes, threads, engagement |
| Facebook | Post | `facebook-post.md` | Community content, events, broad reach |
| YouTube | Short | `youtube-short.md` | Vertical short video, repurposed reels |

## Format Selection Rules

1. Match the content pillar to the strongest format for the platform
2. Educational content defaults to carousels (Instagram) or threads (Twitter)
3. Social proof defaults to quote graphics or video testimonials
4. Behind-the-scenes defaults to reels/stories (Instagram) or video (TikTok)
5. Product content defaults to feed posts (Instagram) or demos (all video)
6. Community content defaults to polls, questions, or UGC reposts
