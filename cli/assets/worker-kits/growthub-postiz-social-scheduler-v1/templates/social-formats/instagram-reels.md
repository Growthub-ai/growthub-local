# Instagram Reels Format

## Constraints

```yaml
duration: "15s, 30s, 60s, or 90s"
aspect_ratio: "9:16"
caption_max: 2200
hashtag_recommended: 5-10
audio: "Original or trending audio"
```

## Structure

### Script/Storyboard

1. **Hook (0-3s)** — Visual or verbal hook that stops the scroll
2. **Setup (3-10s)** — Context or problem statement
3. **Value (10-45s)** — Main content delivery
4. **CTA (last 5s)** — Follow, save, share, or comment prompt

### Caption
- Hook line (complements video hook)
- Brief context or additional value
- CTA
- Hashtag block

## Best Practices

- First 3 seconds determine if someone watches or scrolls
- Use text overlays for accessibility (many watch without sound)
- Trending audio boosts discoverability
- Keep cuts fast — 2-3 second clips
- End with a loop-friendly transition for watch time
- Behind-the-scenes and how-to content performs strongest
