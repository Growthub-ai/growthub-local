# Instagram Feed Post Format

## Constraints

```yaml
caption_max: 2200
hashtag_max: 30
hashtag_recommended: 5-15
image_ratio: "1:1 or 4:5"
image_max_size: "30MB"
video_max_duration: "60s (feed video)"
carousel_max_slides: 10
```

## Structure

1. **Hook** — First line visible before "...more" (~125 chars). Must stop the scroll.
2. **Body** — Value, story, or proof. Use line breaks for readability.
3. **CTA** — Clear action that matches the post objective.
4. **Hashtags** — In caption or first comment. Mixed tier sizes.

## Best Practices

- First line is everything — test multiple hook variants
- Use line breaks and spacing for scannability
- Emojis as bullet points, not decoration
- Tag relevant accounts when appropriate
- Alt text on every image for accessibility
- Save hashtags for first comment to keep caption clean (optional)
