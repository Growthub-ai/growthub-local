# Twitter/X Post Format

## Constraints

```yaml
tweet_max: 280
long_post_max: 25000
hashtag_recommended: 1-2
image_ratio: "16:9 or 1:1"
thread_max: "25 tweets"
```

## Structure

### Single Tweet
1. **Hook + value** — Everything in 280 characters. Punchy, complete thought.
2. **CTA** — Optional. Retweet, reply, or bookmark prompt.

### Thread
1. **Tweet 1 — Hook** — The reason someone should read the thread.
2. **Tweets 2-N — Value** — One idea per tweet. Each should stand alone.
3. **Final tweet — CTA** — Summary + follow/retweet prompt. Link back to tweet 1.

### Caption (with media)
- Short text that complements the image or video
- 1-2 hashtags maximum
- Mention relevant accounts

## Best Practices

- Shorter tweets (under 100 chars) get more engagement
- Threads should be 5-12 tweets for optimal completion rate
- Number your thread tweets (1/, 2/, etc.)
- Use reply-to-self threading, not the compose thread feature
- Hot takes and specific opinions outperform generic advice
- Quote tweets with added insight build authority
- Timing matters — post during commute hours and lunch
