# Facebook Post Format

## Constraints

```yaml
post_max: 63206
optimal_length: "80-100 characters for engagement"
hashtag_recommended: 1-3
image_ratio: "1.91:1 or 1:1"
video_max_duration: "240min"
```

## Structure

1. **Hook** — Question, bold statement, or relatable observation.
2. **Body** — Value or story. Keep paragraphs short.
3. **CTA** — Engagement prompt (comment, share, react).
4. **Hashtags** — 1-3 at the end. Minimal.

## Best Practices

- Short posts (under 100 chars) get highest engagement
- Questions drive comments
- Native video outperforms YouTube links
- Facebook Groups posts get more reach than Page posts
- Use Facebook's built-in poll and event features
- Share-worthy content (lists, tips, relatable observations) spreads organically
- Post between 1-4pm on weekdays for broad audiences
