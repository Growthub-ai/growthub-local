# YouTube Short Format

## Constraints

```yaml
duration: "60s max"
aspect_ratio: "9:16"
title_max: 100
description_max: 5000
hashtag_recommended: 3-5
```

## Structure

### Script/Storyboard

1. **Hook (0-2s)** — Immediate visual or text hook.
2. **Value (2-50s)** — Main content. Tutorial, tip, story, or trend.
3. **CTA (last 5-10s)** — Subscribe, like, or watch full video prompt.

### Title
- Short, keyword-rich, curiosity-driven
- Under 50 characters for mobile visibility

### Description
- Brief context
- Link to full video if applicable
- Hashtags including #Shorts

## Best Practices

- YouTube Shorts algorithm favors watch-through rate — keep it tight
- Repurpose TikTok and Reels content with minor adaptations
- Add #Shorts hashtag for algorithm classification
- Vertical format only — horizontal will not be classified as a Short
- First frame matters — it is the thumbnail in the Shorts feed
- Loop-friendly endings increase watch time
