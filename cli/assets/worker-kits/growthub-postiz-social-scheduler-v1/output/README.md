# Output Directory

All agent-generated deliverables are written here, organized by client and campaign.

## Structure

```
output/
  <client-slug>/
    <campaign-slug>/
      campaign-brief.md
      content-calendar.md
      post-drafts-instagram.md
      post-drafts-linkedin.md
      post-drafts-twitter.md
      caption-matrix.md
      hashtag-matrix.md
      qa-checklist.md
      platform-ready-execution-handoff.md
```

## Usage

The agent creates this structure automatically when it writes its first deliverable. You do not need to create folders manually.

Use the `<client-slug>` to match your brand kit name (e.g. `growthub`, `acme-corp`) and the `<campaign-slug>` to identify the campaign (e.g. `q2-product-launch`, `summer-engagement`).

## Tracking deliverables

Each completed package is logged in the active brand kit under `Deliverables Log`:

```
- YYYY-MM-DD | Postiz Social Package v<N> — <Campaign Name> | output/<client-slug>/<campaign-slug>/
```
